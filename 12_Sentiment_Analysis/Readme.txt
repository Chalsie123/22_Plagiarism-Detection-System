Sentiment analysis (SA) tells user whether the information about the product is satisfactory or not before they buy it. Marketers and firms use this analysis data to understand about their products or services in such a way that it can be offered as per the users requirements. 

We have used jupyter notebook and streamlit for this project. In this project, we firstly preprocess the dataset and split the dataset into two parts for the purpose of training and testing. In training, we use the logistic regression algorithm.
